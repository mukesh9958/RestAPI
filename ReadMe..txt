-----------------This is a Shopping Cart Rest API (in Flask and Mongodb)----------------


>>>>>>>>>>>>>>>>>>>>> This Rest API include following operations:
  	

	1. (PUT) Add product to database.

	2. (GET) Show all products in database.

	3. (GET) Show particular product from database.

	4. (DELETE) Delete particular product from database.

	5. (UPDATE) Update particular product in database.
